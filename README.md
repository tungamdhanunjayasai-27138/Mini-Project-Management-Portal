# Mini Project Management Portal

A modern **Full Stack Task Management Application** built using **React, Node.js, Express, and MySQL**. The application enables users to securely create, manage, organize, and track project tasks through a responsive user interface backed by RESTful APIs and a relational database.

This project demonstrates end-to-end full-stack development, including frontend engineering, backend API development, database design, authentication, and scalable application architecture.

---

# Project Overview

The application provides a simple yet powerful platform for task management with features such as task creation, status tracking, filtering, search, and analytics.

The project was initially developed with essential task management capabilities and was later enhanced with advanced functionality—including secure authentication, pagination, sorting, search, and dashboard statistics—to create a more production-ready solution and demonstrate broader full-stack development skills.
